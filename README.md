# Co-occurrences-in-Terence
This project presents codes and results for the analysis of the co-occurrence of modal markers in a corpus of Terence's comedies.
Reference paper: Marongiu, Paola. "Modal markers in co-occurrence: a study on Terence's comedies." Travaux neuchâtelois de linguistique 79 (2024): 36-67. https://doi.org/10.26034/ne.tranel.2024.4823
